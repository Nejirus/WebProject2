# Daniel's Shopping List Website

### Project Summary
This website is supposed to help keep track of how much you're spending while shopping without having to do the calculations yourself.

### List of Features
You can add the name of an item, its price, how much it's discounted, the quantity, and if it's taxable. The website does all the calculations for you as you put the numbers in. After you submit, you get a receipt of your purchases. Your last submitted data is saved next time you open the website.
